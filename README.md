# SmartStart React Demo

Modern React + TypeScript demo project inspired by a landing page generation workflow.

This project demonstrates a portfolio-ready frontend architecture for a SmartStart-style system where users can choose a landing page type, enter business information, select visual settings and preview a structured landing page concept.

## Project Purpose

This repository is created as a public portfolio demo.

It uses demo data only and does not include production business logic, AI prompts, private automation logic, real client data or the full SmartStart system.
